# 项目简介

## 核心功能
- 基于YOLOv8的番茄实例分割检测系统
- 支持800张标注数据集，包含6类番茄状态
- 支持图片、视频及摄像头进行检测
- 支持自定义置信度、IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 实时显示目标位置、目标总数、置信度等信息

## 适用场景
- 番茄种植质量评估
- 农业自动化检测
- 农产品分类研究
- 计算机视觉教学实验

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9、OpenCV、PyQt5、Ultralytics、Torch1.9等技术栈
- 已训练好模型，配置成功环境可直接使用
- 提供完整源码、标注数据集、训练好的模型及结果、项目运行教程
